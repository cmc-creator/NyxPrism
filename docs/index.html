Quick Start

Install with pip and run your first AI split immediately — no account, no API key required.

quick start
# 1. Install
$ pip install nyxprism

# 2. (Optional) Add your OpenAI key for LLM-quality results
$ export OPENAI_API_KEY="sk-..."

# 3. AI-split a batch scan
$ nyxprism ai-split batch.pdf --output-dir ./split/

# 4. Batch-rename every PDF in a folder
$ nyxprism ai-rename *.pdf --dry-run

# 5. Summarize a document
$ nyxprism ai-summarize report.pdf

# 6. Extract structured info as JSON
$ nyxprism ai-extract-info invoice.pdf
Common questions

FAQ

Everything you need to know before installing.

Do I need an OpenAI API key to use the AI features?

No. Every AI command has a built-in heuristic engine that runs instantly with no API key and no cost. Set OPENAI_API_KEY to unlock LLM-quality results. The default --strategy auto picks the best available option automatically.

Is NyxPrism free to use?

Yes — NyxPrism™ is free and open source under the MIT License. You can use it in personal and commercial projects at no cost. The only potential cost is your own OpenAI API usage if you choose to enable the LLM strategy.

Does NyxPrism send my PDF content to any server?

Only if you explicitly enable the LLM strategy. In that case, text extracted from your PDF is sent to the OpenAI API for analysis. In heuristic mode, all processing is entirely local. No data is ever sent to NyxCollective LLC servers. See our Privacy Policy for full details.

What Python version is required?

Python 3.10 or higher. NyxPrism uses modern union type hints and other Python 3.10+ features throughout the codebase.

Is Tesseract OCR required?

Only if you use the ocr command or AI features on image-only scanned PDFs. Install Tesseract separately (sudo apt install tesseract-ocr on Linux, brew install tesseract on macOS). For PDFs with a selectable text layer, OCR is not needed.

Can I use NyxPrism as a Python library, not just CLI?

Yes. Every command has a corresponding Python function. Import from nyxprism.core.* and nyxprism.ai.* to integrate directly into your own scripts and applications.

How does the --batch-dir mode work for ai-split?

Pass --batch-dir ./folder/ instead of a single file. NyxPrism iterates every *.pdf in the directory, runs the full split pipeline on each one, and places each file’s outputs into its own subdirectory under --output-dir.
