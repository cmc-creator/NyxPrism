Get started in seconds

Quick Start

Install with pip and run your first AI split immediately.

# Install
$ pip install nyxprism

# (Optional) Add your OpenAI key for LLM-quality results
$ export OPENAI_API_KEY="sk-..."

# AI-split a batch scan — works without a key too!
$ nyxprism ai-split batch.pdf --output-dir split_docs/

# Batch-rename every PDF in a folder
$ nyxprism ai-rename *.pdf --dry-run

# Summarize a document
$ nyxprism ai-summarize report.pdf

# Extract structured info as JSON
$ nyxprism ai-extract-info invoice.pdf
