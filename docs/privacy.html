Privacy Policy

NyxCollective LLC  ·  Effective Date: April 24, 2026  ·  Last Updated: April 24, 2026

Short version: NyxCollective LLC does not collect any personal data from you. When you run NyxPrism™, all processing happens on your machine. The only exception is if you choose to enable the optional LLM strategy, which sends document text to the OpenAI API under your own account.

1. Who We Are

This Privacy Policy describes how NyxCollective LLC (“Company,” “we,” “us,” or “our”) handles information in connection with NyxPrism™ (the “Software”) and this website at https://cmc-creator.github.io/NyxPrism/ (the “Site”).

2. Information We Do NOT Collect

NyxCollective LLC does not operate any servers, databases, or backend infrastructure in connection with the Software. We therefore do not collect, store, receive, or process any of the following when you use the Software:

  • PDF documents, file contents, or metadata
  • Extracted text, AI-generated summaries, or analysis results
  • File names, directory paths, or file system information
  • Your name, email address, or any other personally identifiable information
  • Usage telemetry, crash reports, or analytics data
  • IP addresses or device identifiers from Software usage

3. Local Processing — Heuristic Mode

When you use NyxPrism™ in heuristic mode (the default when no OpenAI API key is present), all processing is entirely local to your machine. No document content, metadata, or results are transmitted anywhere. The Software runs fully offline in this mode.

4. Optional LLM Mode — OpenAI API

When you configure an OpenAI API key and use --strategy llm or --strategy auto, the Software will extract text from your PDF documents and transmit it to the OpenAI API for analysis.

This data transmission occurs directly between your machine and OpenAI. NyxCollective LLC never sees, intercepts, or retains this data. The transmission is governed entirely by OpenAI’s Privacy Policy and Terms of Use.

Key points to understand about LLM mode:

  • You provide your own OpenAI API key — NyxCollective LLC has no access to it
  • Any API costs are charged directly to your OpenAI account
  • You are responsible for ensuring you have the right to submit document content to OpenAI
  • Sensitive or confidential documents should use heuristic mode to remain fully local
  • We recommend reviewing OpenAI’s API data usage policies before processing confidential data

5. This Website (GitHub Pages)

This Site is hosted on GitHub Pages, a service of GitHub, Inc. (a subsidiary of Microsoft Corporation). When you visit this Site, GitHub may collect:

  • Your IP address and browser information (server logs)
  • Referrer information and other standard HTTP request data

This data is collected by GitHub, not by NyxCollective LLC. It is governed by GitHub’s Privacy Statement. NyxCollective LLC has no access to GitHub Pages server logs.

6. Cookies and Tracking

NyxCollective LLC does not place any cookies, tracking pixels, analytics scripts, advertising identifiers, or similar technologies on this Site. We do not use Google Analytics, Mixpanel, or any other analytics service.

Google Fonts are loaded via CDN on this Site. This causes your browser to make a request to Google’s servers, which may involve transmission of your IP address to Google. This is governed by Google’s Privacy Policy.

7. PyPI (Python Package Index)

NyxPrism™ is distributed through the Python Package Index (PyPI). When you run pip install nyxprism, your pip client connects to PyPI servers. PyPI is operated by the Python Software Foundation and governed by their Privacy Policy. NyxCollective LLC does not operate PyPI and does not receive data from this interaction.

8. Children’s Privacy

The Software and this Site are not directed at children under 13 years of age. We do not knowingly collect personal information from children. If you believe a child has provided personal information, please contact us so we can address it.

9. Data Security

Because NyxCollective LLC does not collect or store your data, there is no NyxCollective LLC database or server that could be breached to expose your documents or personal information. You are responsible for securing your own files and your OpenAI API key.

10. Your Rights

As NyxCollective LLC does not collect personal data about you, we do not maintain records of you that you would need to access, correct, or delete. If you have questions about data held by GitHub or OpenAI, please contact those services directly.

11. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated “Last Updated” date. We encourage you to review this page periodically. Continued use of the Software or Site after changes are posted constitutes your acceptance of the revised policy.

12. Contact

If you have any questions or concerns about this Privacy Policy, please open an issue on our GitHub repository.
